
#  Air Purifier India - SQL Data Analysis Project

##  Project Overview
This project analyzes India’s air pollution scenario using SQL. It focuses on Air Quality Index (AQI), health impact data,
vehicle registration trends, and population data to generate insights for air purifier market strategy.

The goal is to understand pollution patterns, health risks, and market demand for AirPure Innovations.

##  Objectives
*  Identify major air pollutants affecting Indian cities and states  
*  Analyze AQI trends across time and geography
*  Study relationship between air pollution and health issues
*  Compare weekday vs weekend air quality patterns
*  Estimate market demand for air purifiers using population and vehicle data


##  Datasets Used
- AQI Dataset → Air Quality Index data  
- IDSP Dataset → Health outbreak and disease cases  
- Vahan Dataset → Vehicle registration data  
- Population Projection Dataset → State-wise population data



##  Tools & Technologies Used

- Language: SQL (MySQL)
- CTEs (Common Table Expressions)
- Window Functions (LAG, ROW_NUMBER)
- Aggregations & Joins
- Date Functions


##  Key Analyses Performed
- Top and bottom polluted areas based on AQI  
- Monthly and seasonal pollution trends  
- Weekend vs weekday AQI comparison  
- Health impact correlation with AQI spikes  
- AQI spikes vs vehicle growth analysis  
- Market size estimation using population and pollution burden  



##  Business Impact
- Helps identify high-demand cities for air purifier launch  
- Supports product design based on dominant pollutants  
- Provides insights on health risks linked to pollution  
- Helps estimate market size using real-world data  
- Supports data-driven decision making for AirPure Innovations

 ## Conclusion

This project demonstrates how SQL can be used to analyze real-world environmental and health data to generate actionable business insights for air purifier market strategy in India.
